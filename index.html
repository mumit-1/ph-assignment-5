Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a structured representation of a webpage that allows JavaScript to interact with and manipulate HTML and CSS content dynamically.

How do you select an element from the DOM?

We can select an element from the DOM in various ways. Example: getElementById('id') to select by ID, or querySelector('id/class') to select by CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is when you use one event listener on a parent element to handle events for its child elements, which helps save time and memory.

How do you manipulate an element's attributes and styles using the DOM?

To change an element's attributes and styles, select it and use setAttribute() for attributes and style.property = 'value' for styles.
